# tabla-indian-opus-48

Tabla beats and Indian percussion loops. Sourced from Freesound pack 23020 (DiscordantScraps Broken Drum Machine Tabla Beatz, CC-BY) + one CC0 tabla bol from mmiron.

## About

This repo holds **48 kbps Opus previews** for fast streaming in the web DAW.
Opus at 48k delivers near-CD quality at ~35% the size of MP3 128k.

- **Library**: Tabla / Indian Percussion
- **Files**: 9 Opus previews
- **Bitrate**: 48 kbps VBR
- **Container**: Ogg Opus (.opus)
- **License**: mixed (CC-BY-4.0 + CC0-1.0)
- **Source**: https://freesound.org/people/DiscordantScraps/packs/23020/
- **Branch**: `main`
